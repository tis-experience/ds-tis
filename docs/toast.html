Não use Toast quandoDon't use Toast when
A mensagem deve permanecer no layout (use Alert), a decisão exige foco (use Modal), ou o conteúdo é só label de hover (use Tooltip).The message must stay in the layout (use Alert), the decision needs focus (use Modal), or the content is only a hover label (use Tooltip).

Events: ds-toast-show, ds-toast-dismiss, ds-toast-action.

<script type="module">
  import { initToasts, showToast, destroyToasts } from 'ds-tis/toast';
  initToasts();
  showToast({ type: 'success', title: 'Salvo', description: 'Rascunho atualizado.' });
  showToast({
    type: 'error',
    style: 'solid',
    title: 'Falha',
    actions: [
      { label: 'Tentar de novo', onAction: () => {} },
      { label: 'Detalhes', onAction: () => {} },
    ],
  });
  // destroyToasts() ao desmontar
</script>
